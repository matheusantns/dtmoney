# dtmoney

Projeto criado no curso Ignite da RocketSeat.

O dtmoney é uma aplicação web para controle de finanças pessoais.

## Tecnologias utilizadas

- React;
- Axios;
- MirageJS;
- Polished;
- Styled Components;
- Typescript;
